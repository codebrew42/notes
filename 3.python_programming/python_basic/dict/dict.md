# dictionary

## dic : key - val
### example
- key : val
- bug : an error in program that prevents...
### how it looks
- {key : value}
```python
prog_dic = 
{
	"bug" : "an error in program ....",
	"func" : " a piece of code that you can call...",
	"loop" : "an action of doing ...",
}

print(prog_dic["bug"])
```
